# Sonar Rock vs Mine

## 📌 Problem
This project focuses on a binary classification task: distinguishing between rocks and mines based on sonar signal returns.

##  Dataset
The dataset contains sonar signals reflected from different surfaces. Each sample consists of numerical features representing the energy of the signal at different frequencies.

##  Model
- Logistic Regression

## ⚙️ Approach
- Data preprocessing and conversion to NumPy arrays  
- Train-test split with stratification  
- Model training using Logistic Regression  
- Evaluation on test data  

##  Results
- Test Accuracy: 76%

## 🎯 Conclusion
The model is able to capture patterns in sonar signals and provides a reasonable baseline for this classification task. Further improvements could be achieved using more complex models or feature engineering.
